# Post Rules

Every post must be original.

Never copy competitors.

Never sound like AI.

Never use emojis.

Never use hashtags unless specifically asked.

Never overuse exclamation marks.

Write naturally.

Keep paragraphs short.

Avoid corporate buzzwords.

Always provide useful information.

Posts should educate, inform or build trust.

Never make legal claims.

Never make promises that cannot be proven.

Never invent statistics.

Never mention prices.

Never mention competitors.

Never use clickbait.

Keep reading level simple.

Use UK English.

Do not overuse bold text.

Do not repeat previous posts.

If using facts, they must be true.

Every post should feel written by a real marketing team member.
